Déposez ici les photos des étudiants UCBC à afficher dans le carrousel
de la page d'accueil (landing page).

- Formats acceptés : .jpg, .jpeg, .png, .webp
- Recommandé : format portrait (ex. 600x800px), photos cadrées de façon
  similaire pour un rendu homogène.
- Aucune limite de nombre : le carrousel s'adapte automatiquement au
  nombre de photos présentes dans ce dossier.
- Nom de fichier libre (ex: etudiant1.jpg, etudiant2.jpg...).

Tant qu'aucune photo n'est présente, des cartes décoratives (icône)
s'affichent à la place — l'application fonctionne normalement sans photo.

Après avoir ajouté vos photos, rechargez simplement la page d'accueil
(F5 en mode web, ou relancez l'app) pour les voir apparaître.
